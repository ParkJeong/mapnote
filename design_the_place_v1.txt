앱의 기본사항
    연필 모양, 책 모양, 톱니바퀴 모양 아이콘이 강조된다. EX 카카오톡, 에브리타임

    알림으로는 장소, 제목, 내용이 나타난다. 장소, 제목이 없는 경우 빈칸으로 뜬다.

    알림은 등록된 공간에 진입하면 나타난다. 여러개이면 게시글 중 1개만 뜬다.
    알림은 24시간에 1번만 나타난다. 0시 초기화 또는 알림이 뜨고난 후 24시간 타이머를 재는 방법을 생각 중이다.
    (즉, 알림이 1번 나타나고 나면 등록된 공간을 들락날락해도 알림이 나타나지 않는다. 한번 알림이 나타난 것 만으로 기억이 되살려졌을거라 생각하기 때문이다. 그리고 자주 나타난다면 번거로울 것 같다고 생각했다.)

    등록 안된 곳에 글을 쓰는 경우
        현재 위치 정보를 받아서 반경 50m?를 내부공간으로 등록한다.

    이미 등록된 내부공간에다가 글을 쓰는 경우
        1. 등록된 내부 공간이 1개인 경우
            등록된 내부 공간에다가 글을 등록한다.
        

        2. 등록된 내부 공간이 2개 이상인 경우
            2-1. 등록된 내부 공간들의 원의 중심과 같은 거리에 있는 경우
                같은 거리에 있는 등록된 내부 공간들의 원의 중심과 새로 작성하려는 곳의 거리를 구한다.
                그 거리의 99/100을 새로운 반지름으로 구한다. 
                같은 거리에 있는 등록된 내부 공간들의 원의 중심과 새로 작성하려는 곳을 원의중심으로 하고 아까 새로 구한 반지름을 갖는 원들을 구한다.
                그리고 예전에 등록했던 내부 공간들은 지우고 새롭게 만든 내부공간들을 등록한다.

            2-2. 등록된 내부 공간들의 원의 중심과 다른 거리에 있는 경우

액티비티에서의 활동

2. 글쓰기 화면
    2-1. 글쓰기 화면이 나타나는 조건
        연필 모양 아이콘을 누르는 경우
        1. 로딩화면이 끝나는 경우(앱을 처음 실행하는 경우)

    2-2. 글을 쓰는 장소가 등록된 장소가 아닌 경우
        장소, 제목, 내용칸에서 hint로 장소, 제목, 내용칸인 것을 알려준다.

    2-3. 글을 쓰는 장소가 등록된 장소인 경우
        장소칸에만 이미 등록된 장소 이름이 채워진 상태이다.
        장소칸의 내용을 바꾸게 되면 기존에 등록된 장소이름이 수정된다.(장소이름을 수정하고 저장하는 경우 경고를 한다.)
	
    2-4. 추가 개선 사항
        사진/영상 추가는 카메라 버튼을 눌러서 한다.(추가 구현 예정)
        Q. 사용자가장소의 이름을 2개이상 하고 싶다면? -> 추후에 다시 생각


3. 글 목록 보기
    3-1. 글 목록 보기화면이 나타나는 조건
        책 모양 아이콘을 누르는 경우
  
    아래를 향한 세모버튼을 누르면 정렬기준을 선택할 수 있다. 정렬 기준은 시간순, 오래된 순, 거리가 가까운 순(시간순이 기본값이다.)으로 총 3가지이다.
    돋보기 모양 아이콘을 누르는 경우 6. 게시글 검색 화면으로 넘어간다.
    사진이 있는 경우 사진 중 1개만 미리보기가 제공된다.

    3-2. 추가 개선 사항
        가능하면 슬라이드 형식으로 구현
        옆으로 슬라이드나 인스타그램처럼 위아래로 슬라이드하는 형식
        만약 이렇게 구현한다면 5. 게시글 보기화면은 없앤다.
        현재 생각한 게시글을 보는 방법은 1. 게시글 목록과 보기를 분리한다. 2. 게시글 목록과 보기를 분리 후 옆 슬라이드식으로 본다. 3. 게시글 목록과 보기를 분리 후 위아래 슬라이드식으로 본다. 3가지가 있다. 일단 1번으로 구현하되 차후에 2,3번을 구현 후 비교해볼 예정이다.

  
4. 설정
    4-1. 설정 화면이 나타나는 조건
        톱니바퀴아이콘을 누르는 경우

    내 정보를 확인하고 변경할 수 있다.
    나중에 서버에도 기록을 저장할 계획이다. 이를 위해 사용자를 식별할 아이디가 필요하다.
    현재는 핸드폰에만 저장하도록 구현할 예정

    알림설정을 변경할 수 있다. 소리, 진동, 무음


5. 게시글 보기
    5-1. 게시글보기 화면이 나타나는 조건
        3. 글 목록 보기 화면에서 게시글 클릭을 하는 경우
        7. 게시글 검색 결과 화면에서 게시글을 클릭한 경우

    뒤로가기 모양 버튼 옆에 장소/시간이 적혀 있다.
    장소이름이 없는 경우 시간만 적혀있다.
    시간은 년 월 일로 적혀있다.

6. 게시글 검색
    뒤로가기 모양 버튼을 누르면 3. 글 목록 보기 화면으로 돌아간다.
    검색창에는 장소, 글 제목, 내용을 hint로 알려준다.
    해당 검색어를 포함한(장소, 글 제목, 내용) 모든 게시글이 나타난다.

7. 게시글 검색 결과
    7-1. 게시글 검색 화면이 나타나는 조건
        7-1-1. 6. 게시글 검색에서 넘어온 경우
            뒤로가기 모양 버튼 옆에 검색어가 적힌다.
    
        7-1-2. 알림을 누르고 넘어온 경우
            뒤로가기 모양 버튼 옆에 장소이름이 적힌다.
            장소이름이 없는 경우 빈칸으로 놔둔다.
            해당 장소에 저장된 모든 게시글이 나타난다.

    뒤로가기 모양 아이콘을 클릭하면 3. 글 목록 보기 화면으로 넘어간다.

        5. 게시글 보기에서 수정 버튼을 누르는 경우
